Mis proyectos del módulo 2:

Introducción a CSS

En este proyecto se exploraron principios básicos de CSS, como selectores, propiedades, pseudo-clases y pseudo-elementos.

Se exploraron diferentes maneras de disponer un texto, así como su alinecaión, estilos, colores y fuentes.

Box model / Display

En esta ocasión nos adentramos al box model y el display.

Entendimos cómo funcionan los valores de display (block, inline e inline-block) y su comportamiento en el box model.

Especificidad y herencia

En este proyecto exploramos cómo funcionan las especificidades y herencias en un código CSS.

Si bien no es un archivo con demasiados elementos, fue una clase en la que se le dedicó mayor enfoque al archivo CSS para entender como funciona la herencia y la especificidad.

FlexBox

En este proyecto exploramos el uso de FlexBox para organizar y alinear elementos en una página web.

Además se exploraron las propiedades de FlexBox para saber cómo podemos disponer los elementos de nuestro sitio web a como necesitemos hacerlo.

Adicionalmente, aprendimos a configurar un archivo de JS para lograr que cambiemos de un tema claro a un tema oscuro y viceversa.

Simulación página de ecommerce

En esta clase utilizamos las funciones de FlexBox para crear una página de ecommerce que imitara a una tienda online de ropa deportiva.

Para la generación de imágenes utilizamos un sitio lorem ipsum, pero dedicado a generar imágenes aleatorias con el fin de explorar la funcionalidad de FlexBox y lograr la simulación de una página de ecommerce de este estilo.

Simulación página tipo Pinterest

En esta ocasión trabajamos las posiciones de FlexBox para crear un sitio web con un estilo similar a Pinterest, en el cual al pasar por encima el cursor de una imagen aparece un sombreado que revela un texto o un enlace a otra página.
