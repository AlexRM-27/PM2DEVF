Exploranding el box model y el display

Esta página muestra cómo reacciona el modelo de cada elemento con diferentes valores de display y box-sizing

Display: Block

Los elemenentos con display:block ocupan el ancho completo disponible y empiezan en una nueva línea.

Soy otro elemento de bloque.

Display: Inline

Los elementos con display:Inline solo ocupan el ancho necesario y no aceptan propiedades de tamaño como width y height.

Soy inline. Soy otro inline.

Display: Inline-Block

Los elementos con display:Inline-Block combinan las características de inline (no rompen línea) con block (aceptan propiedades de width y height)

Soy inline-block. Soy otro inline-block.

El box model en acción.

El box model afecta el espacio y diseño de cada elemento según su display.

Display: Block

Cada elemento de tipo block ocupa el ancho completo y se apilan uno debajo del otro.

El padding, border y margin funcionan como se esperan.

Bloque 2

Display: Inline

Los elementos inline se alinean horizontalmente como texto.

El margin horizontal (izquierda a derecha) funciona, pero el vertical (arriba a abajo) no afecta el flujo del documento. Es decir, se ignoran estas propiedades.

Display: Inline-Block

Los elementos inline-block se alinean hgorizontalmente como inline, pero permiten usar width, height, padding, border y margin como los elementos de display block.
